

# Лабораторная работа №8: Адаптивная верстка с использованием CSS Grid

## Введение

Цель этой лабораторной работы — освоить современные технологии CSS для построения адаптивных макетов веб-страниц с помощью CSS Grid. В ходе работы вы научитесь создавать гибкие сетки, которые автоматически подстраиваются под размер экрана: будь то компьютер, планшет или смартфон.

Адаптивная верстка — это важнейший аспект современного веб-дизайна, позволяющий обеспечить комфортное взаимодействие пользователей с сайтом на различных устройствах.

## Таблица сравнения Flexbox и CSS Grid

| Критерий                     | Flexbox                   | CSS Grid                |
|------------------------------|---------------------------|-------------------------|
| Тип компоновки               | Одномерная                | Двумерная              |
| Управление                   | Строка или столбец       | Строки и столбцы      |
| Основное назначение          | Выравнивание элементов     | Построение макетов     |
| Подходит для                 | Компонентов               | Страниц и сеток       |



## Типичные сценарии использования                                                                                            
| Задача                      | Рекомендуется       |
|-----------------------------|----------------------|
| Навигационное меню         | Flexbox              |
| Карточки товаров            | Flexbox + Grid       |
| Центрирование элемента      | Flexbox              |
| Макет страницы              | Grid                 |
| Галерея изображений         | Grid                 |
| Дашборд                     | Grid                 |

*Обратите внимание:* В большинстве случаев Flexbox лучше подходит для линейных задач, таких как меню или карточки, тогда как CSS Grid отлично справляется с более сложными макетами и сетками.
